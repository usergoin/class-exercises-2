Java Exercises Project 

This repository contains a set of Java programs developed to practice programming logic and problem-solving using arrays and basic algorithms.

📌 Overview

The project includes multiple exercises that demonstrate fundamental concepts such as:
Array manipulation
Mathematical calculations
Data analysis
Search algorithms
Vector operations

⚙️ Implemented Features

Each class represents a different exercise:
Ado.java → Calculates class average and highest grade
questao3.java → Sum of positive numbers and count of negative numbers
questao4.java → Search for a number inside an array
questao5.java → Calculates average and filters values above the average
questao6.java → Calculates the dot product between two vectors
questao7.java → Calculates student averages and number of approved students
questao10.java → Normalizes values based on the largest number

🛠️ Technologies
Java
Arrays and loops
Scanner (user input)
